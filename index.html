📒 Supabase instellen (eénmalig)
1Account & project aanmaken
Ga naar supabase.com → Sign up → New project → naam: heli-crm → regio: West EU (Frankfurt)
2Tabellen aanmaken via SQL Editor
Ga naar SQL Editor in Supabase → plak dit en klik Run:
CREATE TABLE IF NOT EXISTS leads (
  id TEXT PRIMARY KEY,
  bedrijf TEXT, groep TEXT, postcode TEXT,
  "sapNr" TEXT, contact TEXT, email TEXT, tel TEXT, werf TEXT,
  bu TEXT, assign TEXT, status TEXT, waarde NUMERIC,
  prio TEXT, omschr TEXT, partners TEXT,
  datum DATE, "nextContact" DATE,
  "doorgestuurdDoor" TEXT, "csVan" TEXT,
  interacties TEXT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS sap_klanten (
  nr TEXT PRIMARY KEY,
  naam TEXT, bus TEXT,
  omz26 NUMERIC DEFAULT 0,
  omz25 NUMERIC DEFAULT 0,
  fac INTEGER DEFAULT 0,
  ord INTEGER DEFAULT 0
);
3URL en API-sleutel kopiëren
Ga naar Project Settings → API → kopieer Project URL en anon public key → plak ze hierboven → klik "Opslaan & verbinden"
4Excel importeren
Sleep de Excel-bestanden naar de vakjes hierboven → klik "Importeren & opslaan in database" → klaar. Voortaan beheert de CRM alles zelf.
